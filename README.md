# STAR
STAR(Summarize Text and Automate Recitatioin)は、Notionの記事のデータベースから、記事を内容を要約してAIに読み上げさせた音声ファイルをGoogleDriveに保存してPodcastとして配信します。  
必要なもの  
・OpenAI-API-key　　
・Diffbot-APIトークン  
・GoogleDriveのフォルダID
・Notion-APIトークン  
・NotionデータベースID  
・token.json(GoogleDriveAPI)  
・credential.json(GoogleDriveAPI)
